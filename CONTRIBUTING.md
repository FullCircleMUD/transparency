# Contributing

This repository contains FullCircleMUD's financial
reports and compliance documentation.

**To report an error in a published document:**
Open an issue describing the error and the correct
information.

**For compliance or regulatory enquiries:**
Contact compliance@fullcirclemud.com

**For general questions:**
See fullcirclemud.com or join the community at
[discord/forum link].

Pull requests are not accepted for financial reports
(published records must not be altered) but are
welcome for the README and supporting documentation.
